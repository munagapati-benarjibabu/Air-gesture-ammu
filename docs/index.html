Air Gesture Ammu

Android voice and air-gesture assistant using CameraX, MediaPipe Hands, Accessibility Service actions, speech commands, sensor controls, and lost-phone tracking.

Download PPT Open PDF Report View Source

Project Features

Air Gestures

Detects swipe, open palm, and fist gestures from the front camera and maps them to phone actions.

Voice Assistant

Uses Android speech recognition and text-to-speech for Ammu assistant commands.

Accessibility Control

Runs scroll, back, home, recent apps, and media controls through Android services.

Lost-Phone Tracking

Sends scheduled location email updates when the configured recovery mode is activated.

Run The Project

  • Open the project in Android Studio with JDK 17 or newer.
  • Connect an Android phone with USB debugging enabled.
  • Build with ./gradlew :app:assembleDebug.
  • Install with ./gradlew :app:installDebug.
  • Enable microphone, camera, location, notification, and accessibility permissions on the phone.
